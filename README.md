# CC2PDF

A macOS SwiftUI application for converting files to PDF.

## Requirements

- macOS 14+
- Xcode 15+ or Swift 5.9+

## Build

```
swift build
```

## Test

```
swift test
```

## Usage

1. Launch the app.
2. Click **Choose…** to select an input file.
3. Click **Convert to PDF** to convert.

The output PDF is written alongside the input file.
